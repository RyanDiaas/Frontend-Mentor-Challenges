## 🚀 Sobre o Projeto

Este projeto consiste no desenvolvimento de uma **landing page responsiva** baseada em um layout moderno.

O objetivo foi praticar conceitos fundamentais de **desenvolvimento front-end**, com foco em:

* Estruturação semântica com **HTML5**
* Estilização com **CSS3**
* Desenvolvimento **Mobile First**
* Layouts responsivos com **Flexbox**
* Uso de **variáveis CSS**
* Tipografia responsiva com **clamp()**

A landing page apresenta uma interface para divulgação de um aplicativo chamado **Clipboard**, que permite armazenar e organizar conteúdos copiados entre diferentes dispositivos.
