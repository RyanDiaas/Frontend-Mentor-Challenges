# Product Preview Card

Projeto desenvolvido como prática de **HTML e CSS** a partir de um desafio da plataforma Frontend Mentor.
O objetivo foi recriar o layout proposto com a maior fidelidade possível, trabalhando principalmente **estrutura semântica, layout com Flexbox, tipografia e responsividade**.

## 📸 Preview

Layout de um **card de produto**, contendo:

* imagem do produto
* categoria
* título
* descrição
* preço atual e preço antigo
* botão de adicionar ao carrinho

O design foi implementado seguindo o *style guide* fornecido no desafio.

## 🚀 Tecnologias utilizadas

* HTML5
* CSS3
* Flexbox
* Media Queries
* Google Fonts

## 📱 Responsividade

O layout foi adaptado para diferentes tamanhos de tela utilizando **media queries**, mudando a estrutura do card de **layout horizontal (desktop)** para **layout vertical (mobile)**.

## 🎯 Objetivos do projeto

Este projeto foi desenvolvido para praticar:

* Estruturação semântica em HTML
* Organização de CSS
* Uso de variáveis CSS
* Centralização de elementos com Flexbox
* Controle de espaçamento e tipografia
* Criação de layouts responsivos

Durante o desenvolvimento foram aplicados conceitos importantes como:
