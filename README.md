# Frontend Mentor Challenges

Este repositório reúne uma coleção de desafios realizados da plataforma **Frontend Mentor** com o objetivo de praticar **HTML, CSS e desenvolvimento front-end**.

Os projetos foram desenvolvidos seguindo os layouts propostos nos desafios, buscando reproduzir o design com a maior fidelidade possível.

---

## 🎯 Objetivo

O principal objetivo deste repositório é:

* Praticar **HTML semântico**
* Melhorar habilidades de **CSS e layout**
* Aplicar conceitos de **Mobile First**
* Aprender **Flexbox e Grid**
* Desenvolver atenção a **detalhes visuais**
* Construir um **portfólio de projetos front-end**

---

## 🚀 Desafios concluídos

* Social Links Profile
* QR Code Component
* Recipe Page
* Product Preview Card Component
* Stats Preview Card Component
* Social Proof Section

---

## 🛠 Tecnologias utilizadas

Os projetos foram desenvolvidos utilizando:

* HTML5
* CSS3
* Flexbox
* CSS Grid
* Mobile First
* Google Fonts

---

## 📂 Estrutura do repositório

Cada desafio possui sua própria pasta contendo os arquivos do projeto.

Cada pasta contém:

* `index.html`
* `style.css`
* assets do desafio

Frontend Mentor é uma plataforma que fornece **desafios reais de front-end**, permitindo que desenvolvedores pratiquem a construção de interfaces com base em designs profissionais.
